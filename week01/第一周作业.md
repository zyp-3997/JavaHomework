
## 1.（选做）自己写一个简单的 Hello.java，里面需要涉及基本类型，四则运行，if 和 for，然后自己分析一下对应的字节码，有问题群里讨论。
代码
```java
public class Main {
    public static void main(String[] args) {
        int a=1;
        int b=2;
        int c=methosAdd(a,b);
        System.out.println(c);
    }

    private static int methosAdd(int a, int b) {
        if (a==1){
            return a+b;
        }
        else return a+b+1;
    }
}
```
